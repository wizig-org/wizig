# Wizig Android SDK

This module provides shared Android-side Kotlin types used by Wizig apps.

Interop model:

- Android interop is JNI-first and generated by `wizig codegen`.
- Generated app bindings live under `.wizig/generated/kotlin/dev/wizig`.
- Generated JNI bridge lives under `.wizig/generated/android/jni`.

Notes:

- `WizigRuntime` in this SDK is a compatibility shim and does not perform
  direct FFI calls on Android.
- Apps should call generated JNI-backed `WizigGeneratedApi` surfaces.

Generate registrants:

```sh
zig build run -- plugin sync .
```
